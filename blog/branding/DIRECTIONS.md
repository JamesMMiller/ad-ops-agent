# Evening Ops — brand directions

Public Hashnode build diary for James Miller: evenings after a lead-developer day job, documenting Our Tech Accessories and the ad-ops-agent experiment.

**Publication:** Evening Ops · [eveningops.hashnode.dev](https://eveningops.hashnode.dev)

Related to **Our Tech** (store), but not the same mark. Store keeps the circular **OT**. Evening Ops is the after-hours journal.

## Selected direction: **A — EO squircle + amber bar**

| Piece | Spec |
|-------|------|
| **Mark** | Soft squircle containing an **EO** ligature (near-black). Thin warm **amber** horizontal accent (desk-lamp / after-work cue). |
| **Wordmark** | Stacked `EVENING` / `OPS`, all caps, geometric sans, open tracking. |
| **Lockup** | Mark left + stacked wordmark right (horizontal). |
| **Favicon** | EO monogram only (or squircle mark cropped tight). |
| **OG / social** | Dark charcoal field, mark + `EVENING OPS` + short line: *Build diary from after work.* |

**Alt explored (not primary):** **B** — circular stacked E / amber rule / O. Cleaner, less distinctive; keep as fallback.

## Colour

| Token | Hex | Use |
|-------|-----|-----|
| Ink | `#0A0A0A` | Primary type + marks on light |
| Paper | `#FFFFFF` | Light backgrounds |
| Dusk | `#121212` | Dark surfaces / OG |
| Lamp | `#C9853A` | Accent only (bar, sparse UI highlights) |
| Mute | `#6B6B6B` | Subtitles, meta |

Avoid purple neon, crypto gradients, and carnival orange.

## Typography

- **Display / wordmark:** Geometric sans, all caps, slightly open tracking (think premium tech journal, not Inter UI chrome).
- **Body (Hashnode):** System / publication default is fine; prefer readable sans. UK English in prose.
- No script fonts. No heavy slab “broadsheet” looks.

## Voice (one line)

Calm engineer after work. Dry humour. Honest ops. Not a hustle brand.

## Relationship to Our Tech

| | Our Tech Accessories | Evening Ops |
|--|----------------------|-------------|
| Job | Sell calm tech | Tell the build story |
| Mark | Circular **OT** | Squircle **EO** + amber |
| Domain | ourtechaccessories.com | eveningops.hashnode.dev |
| Tone | Restrained retail | Diary / after-hours |

Do not swap marks between store and blog.
